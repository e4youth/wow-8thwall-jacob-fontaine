# WOW — 8th Wall Jacob Fontaine (Monorepo)

What Once Was AR Heritage Tour — stop-specific 8th Wall / A-Frame experiences for Jacob Fontaine, plus shared components.

## Structure
- `experiences/<stop>/` — a standalone 8th Wall project (app.js, body.html, index.css, assets, build config)
- `shared/` — shared A-Frame components and styles we will gradually consolidate

## Build
Each experience currently builds independently using its own `package.json`.

## Notes
